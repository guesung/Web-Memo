# 작업 문서화 - Notion 페이지 생성

현재 세션에서 수행한 작업을 Notion "웹 메모 - AI 문서화" 데이터베이스에 기록합니다.

## 수행 절차

### 1. 현재 작업 내용 파악

아래 명령어들을 병렬로 실행하여 현재 작업 상태를 파악하세요:
- `git diff --stat` - 변경된 파일 목록
- `git diff` - 변경 내용 상세
- `git log --oneline -10` - 최근 커밋 히스토리
- `git branch --show-current` - 현재 브랜치

### 2. Notion 페이지 생성

`mcp__claude_ai_Notion__notion-create-pages` 도구를 사용하여 페이지를 생성하세요.

**설정값:**
- `parent`: `{"data_source_id": "30b89de0-2fde-8088-aa3f-000ba0277d43"}`

**속성 (properties):**
- `이름`: 작업 제목 (예: "사이드바 카테고리 필터 구현")
- `date:날짜:start`: 오늘 날짜 (ISO-8601, 예: "2026-02-18")
- `date:날짜:is_datetime`: 0
- `요약`: 작업 내용을 1-2문장으로 요약

**본문 (content):**
아래 구조로 작업 내용을 상세히 기록하세요:

```
## 변경 사항
- 변경된 파일별로 어떤 수정이 이루어졌는지 상세 기술

## 기술적 세부사항
- 사용된 기술, 패턴, 라이브러리
- 주요 구현 로직 설명

## 관련 컨텍스트
- 브랜치명, PR 번호 등
- 관련 이슈나 배경 설명
```

### 3. 결과 보고

생성된 페이지의 URL을 사용자에게 공유하세요.
